# Code de conduite

## Notre engagement
Nous voulons une communauté **ouverte, accueillante et bienveillante** pour tous,
quel que soit le niveau, l'âge, l'origine, le genre ou l'expérience. C'est un projet
d'apprentissage : les débutants sont les bienvenus, les questions « bêtes » n'existent pas.

## Attendu
- Être respectueux et constructif (critiquer le code, pas les personnes).
- Accueillir les nouveaux et aider quand on peut.
- Accepter les retours avec ouverture.

## Inacceptable
- Harcèlement, insultes, propos discriminatoires.
- Trolling, attaques personnelles, spam.
- Partager des données privées d'autrui.

## Application
Les mainteneurs peuvent éditer/supprimer les contributions non conformes et bannir
en cas d'abus répété. Pour signaler un problème, ouvre une issue privée ou contacte
un mainteneur.

Inspiré du [Contributor Covenant](https://www.contributor-covenant.org/) v2.1.
